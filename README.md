# Simple Calculator

## Description
A beginner-friendly command-line calculator built with Python. It performs basic arithmetic operations with a simple menu interface.

## Features
- Addition (+)
- Subtraction (-) 
- Multiplication (*)
- Division (/)
- User input validation
- Menu-driven program

## How to Run
1. Download `CALCULATOR.py`
2. Open terminal/command prompt
3. Run: `python CALCULATOR.py`
4. Choose operation and enter numbers

- Python 3
